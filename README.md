# Asteroids-Clone
Asteroids game for Linux written with C++ and SDL

Install SDL2 and SDL2-image development libraries:

```
apt-get install libsdl2-dev
apt-get install libsdl2-image-dev
apt-get install libsdl2-ttf-dev
```

then make.
